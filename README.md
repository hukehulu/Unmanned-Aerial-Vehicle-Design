# Android Drone Ground Station (DroneGCS)

## 项目简介
DroneGCS 是一个基于 **Android SDK** 和 **Java** 开发的无人机安卓端地面站 (GCS) 项目。它针对移动端无人机控制的痛点，提供了完整的地图显示、遥控操控、视频传输和飞控通信解决方案。

---

## 核心痛点与解决方案

1. **移动端无人机控制复杂**
   - 传统无人机控制软件多基于 PC 或专用硬件，移动端功能有限。
   - **解决方案**：DroneGCS 将地面站完整功能移植到 Android 端，集成地图显示、遥杆控制和视频传输，便于操作与部署。

2. **实时图传与飞控通信延迟高**
   - RTSP 视频流与串口指令可能存在延迟和丢包问题。
   - **解决方案**：通过 `ExoPlayer` 或 `VLC` 进行视频缓冲优化；串口通信使用异步线程和缓冲队列确保指令可靠发送。

3. **自定义操作界面不足**
   - 现有 SDK 遥杆控件限制多，无法满足个性化需求。
   - **解决方案**：自行设计 UI 绘制类，实现可定制化遥杆控件，支持多点触控和归中逻辑。

## 核心逻辑流

### 1. 系统模块化设计
DroneGCS 的逻辑可分为四个核心模块：

```text
+------------------+       +------------------+
|   Map Module     |       |  Flight Control  |
|  (高德地图API)   | <---> |  (串口通信协议)  |
+------------------+       +------------------+
          ^                         ^
          |                         |
          v                         v
+------------------+       +------------------+
|  Joystick Module |       |  RTSP Video      |
| (自定义控件)     |       |  Stream Player   |
+------------------+       +------------------+

## 技术栈
- Java + Android SDK
- 高德地图 Android SDK
- RTSP 视频流解析（使用 `ExoPlayer` 或 `VLC for Android`）
- 串口通信（`UsbSerial` 或 `SerialPort`）

## 运行环境
- Android Studio 2022+
- Android SDK 28+
- 真机调试推荐（模拟器无法访问串口）
